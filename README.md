This project simulates a real-world A/B test on a synthetic dataset representing 
an AI-powered personalized recommendation engine introduced into an educational 
learning platform. The goal is not just to measure lift — but to determine whether 
the feature justifies its inference and API costs before a staged rollout.

Built from research papers on experimentation design and ethical AI deployment, 
the pipeline covers the full decision lifecycle: data cleaning, statistical testing, 
regression modeling, segmentation analysis, and an ethics module — culminating in 
a go/no-go rollout recommendation backed by data.
